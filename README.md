# 🌍 Time Zone Converter

A fun, colorful example project that shows the current time in cities around the world!

## What it does

This program lets you enter a city or country, and it shows you what time it is there right now. It's a great way to stay connected with friends, family, or classmates around the world.

## How to run it

1. Make sure you have Python installed
2. Download `time_zone_converter.py`
3. Run it with: `python time_zone_converter.py`
4. Follow the prompts!
